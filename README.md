# Ya GPT — Яндекс AI для Obsidian

Мощный плагин для Obsidian, интегрирующий [Яндекс AI Studio](https://console.yandex.cloud) прямо в ваш второй мозг.

## Возможности

- 💬 **Чат с YandexGPT** — полноценный чат-интерфейс в боковой панели
- 🤖 **Выбор модели** — YandexGPT Lite, Pro, Pro 32k, Llama
- ⚡ **Потоковый режим** — ответы появляются постепенно (streaming)
- 📝 **Контекст заметки** — автоматически включать текущую заметку в контекст
- 📋 **Вставка в заметку** — одним кликом вставить ответ AI в редактор
- 🎨 **Красивый UI** — тёмная/светлая тема, адаптивный дизайн
- 💾 **История чата** — сохраняется между сессиями
- ⌨️ **Команды** — резюме, улучшение текста, перевод

## Установка

### Ручная установка (разработка)

```bash
# Клонировать репо
git clone https://github.com/logotipza/obsidian-ya-gpt.git

# Перейти в папку
cd obsidian-ya-gpt

# Установить зависимости
npm install

# Собрать плагин
npm run build

# Скопировать в vault
cp -r . /path/to/vault/.obsidian/plugins/obsidian-ya-gpt/
```

### Получение API ключа

1. Зайдите на [console.yandex.cloud](https://console.yandex.cloud)
2. Создайте сервисный аккаунт или используйте свой
3. Создайте API ключ
4. Вставьте ключ в настройки плагина

### Настройка

1. Откройте **Настройки → Плагины → Ya GPT**
2. Вставьте **API Ключ** от Яндекс Cloud
3. При необходимости укажите **Folder ID**
4. Выберите модель и настройте параметры

## Команды

| Команда | Описание |
|---------|----------|
| `Открыть чат Ya GPT` | Открывает панель чата |
| `Спросить Ya GPT о текущей заметке` | Открывает чат с контекстом заметки |
| `Резюме текущей заметки` | Краткое резюме через AI |
| `Улучшить текст` | Улучшить выделенный текст |
| `Перевести выделенное` | Перевод на английский |

## Горячие клавиши

- **Enter** — отправить сообщение
- **Shift+Enter** — новая строка
- **Cmd/Ctrl+Enter** — отправить (альтернатива)
